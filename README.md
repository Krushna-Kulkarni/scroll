- [ ] Home page
  - [x] Feed
  - [x] Post
  - [x] Sort Posts
  - [x] Post Interactions
  - [ ] Post Comments
  - [ ] Post options
  - [ ] Search users
  - [ ] Follow users
  - [x] Create Post
- [x] Bookmarks Page
- [x] Explore Page
- [ ] User Profile Page
  - [x] Bio

  - [ ] Edit Bio and Profile Picture

  - [ ] User Signup
  - [ ] User Login

## **Built with**

- React JS
- TailwindCSS
- React Context API
- React Router v6

## **How to install and run locally?**

```
$ git clone https://github.com/Krushna-Kulkarni/scroll
$ cd scroll
$ npm install
$ npm start
```
